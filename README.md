mkv2mp4
=======
Prerequisites
-------------
* mkvinfo
* mkvextract
* MP4Box

Installation of prerequisites
-----------------------------
Ubuntu Linux systems:
* sudo apt-get install mkvtoolnix
* sudo apt-get install gpac

Usage
-----
* mkv2mp4 [-q] file.mkv
* This command converts file.mkv into file.mp4
* -q is optional for quite operation
